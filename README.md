# Random Password Generator (C++)

## 📌 Overview
A console-based application to generate secure random passwords with customizable options.

## 🚀 Features
- Custom password length
- Optional uppercase letters
- Optional digits and symbols
- Randomized secure generation

## 🛠️ Tech Stack
- C++
- Standard Library

## ▶️ How to Run
1. Compile:
   g++ password.cpp -o password
2. Run:
   ./password
